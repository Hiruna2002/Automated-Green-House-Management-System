#  Automated Green House Management System
   
 Overview

The Automated Green House Management System is an IoT-based smart agriculture solution designed to monitor and control environmental conditions inside a greenhouse automatically.

This system helps optimize plant growth by continuously tracking parameters such as temperature, humidity, and soil moisture, and taking automated actions when necessary.

   Features
   
 - Real-time temperature monitoring
 
 - Soil moisture detection & automatic irrigation
 
 - Humidity control system
 
 - Automated decision making using sensors
 
 - Live data display (LCD / Dashboard)
 
 - Alerts for abnormal conditions
 
 - Low-cost and efficient design

   How It Works
   
- Sensors collect environmental data
- Microcontroller processes data
- If conditions are not optimal:
    - Water pump turns ON/OFF
    - Fan or cooling system activates

- Data is displayed on LCD / dashboard
  
Advantages
-  Improves crop yield
- Saves water with smart irrigation
- Reduces manual work
- Enables automation in agriculture
